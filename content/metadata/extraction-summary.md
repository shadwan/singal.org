## Extraction Format Features

Each markdown file includes:
- **YAML Frontmatter** with metadata (title, category, location, colors, dates, etc.)
- **Exact Content Preservation** - All text exactly as it appears on site
- **Image References** - Both original paths and context/usage notes
- **HTML Structure Notes** - Layout patterns, CSS classes, component structure
- **Impact Metrics** - Numerical data (grant amounts, timelines, people served)
- **Related Content** - Cross-references to other pages/stories
- **Layout Documentation** - Column structure, responsive patterns
